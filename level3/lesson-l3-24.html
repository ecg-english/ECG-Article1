24. 仮定法（過去・過去完了）

もし〜だったら、の表現を完全マスター

仮定法とは何か？

仮定法は、現実とは異なる仮定や、実現可能性が低いことを表す表現です。仮定法過去と仮定法過去完了があります。

仮定法の基本

仮定法の種類

仮定法過去：現在の事実に反する仮定（もし今〜だったら）

仮定法過去完了：過去の事実に反する仮定（もしあの時〜だったら）

仮定法と直説法の違い

直説法：If it rains, I will stay home. （雨が降る可能性がある）

仮定法：If it were sunny, I would go out. （今は晴れていない）

→ 仮定法は現実とは異なることを表す

仮定法過去（If + 過去形）

仮定法過去は、現在の事実に反する仮定を表します。「もし今〜だったら」という意味です。

仮定法過去の基本構造

基本形

If + 主語 + 過去形, 主語 + would/could/might + 動詞の原形

「もし〜だったら、〜だろう」

例文1

If I were rich, I would travel the world.

「もし私がお金持ちだったら、世界中を旅行するでしょう」

→ 現在はお金持ちではない（事実に反する）

例文2

If I had time, I would help you.

「もし私に時間があったら、あなたを助けるでしょう」

→ 現在は時間がない（事実に反する）

wereの使い方

仮定法過去では、主語が何であってもbe動詞はwereを使います（wasではなく）。

wereを使う例

If I were you, I would study harder.

「もし私があなただったら、もっと勉強するでしょう」

If he were here, he would help us.

「もし彼がここにいたら、私たちを助けるでしょう」

→ 仮定法では主語が何でもwereを使う（口語ではwasも可）

would, could, mightの使い分け

would（〜だろう）

If I were rich, I would buy a house.

「もし私がお金持ちだったら、家を買うでしょう」

→ 確実性が高い

could（〜できるだろう）

If I had time, I could help you.

「もし私に時間があったら、あなたを助けることができるでしょう」

→ 可能性・能力を表す

might（〜かもしれない）

If I were you, I might accept the offer.

「もし私があなただったら、その申し出を受け入れるかもしれません」

→ 不確実性が高い

仮定法過去完了（If + 過去完了形）

仮定法過去完了は、過去の事実に反する仮定を表します。「もしあの時〜だったら」という意味です。

仮定法過去完了の基本構造

基本形

If + 主語 + had + 過去分詞, 主語 + would/could/might have + 過去分詞

「もしあの時〜だったら、〜だっただろう」

例文1

If I had studied harder, I would have passed the exam.

「もし私がもっと勉強していたら、試験に合格していたでしょう」

→ 過去にもっと勉強していなかった（事実に反する）

例文2

If I had known that, I would have told you.

「もし私がそれを知っていたら、あなたに伝えていたでしょう」

→ 過去に知らなかった（事実に反する）

would have, could have, might haveの使い分け

would have（〜だっただろう）

If I had studied harder, I would have passed the exam.

「もし私がもっと勉強していたら、試験に合格していたでしょう」

→ 確実性が高い

could have（〜できただろう）

If I had had time, I could have helped you.

「もし私に時間があったら、あなたを助けることができたでしょう」

→ 可能性・能力を表す

might have（〜だったかもしれない）

If I had been there, I might have seen him.

「もし私がそこにいたら、彼を見たかもしれません」

→ 不確実性が高い

仮定法過去と仮定法過去完了の使い分け

仮定法過去と仮定法過去完了の使い分けを理解しましょう。

時制の違い

仮定法過去（現在の事実に反する）

If I were rich, I would travel.

「もし私がお金持ちだったら、旅行するでしょう」

→ 現在はお金持ちではない

仮定法過去完了（過去の事実に反する）

If I had been rich, I would have traveled.

「もし私がお金持ちだったら、旅行していたでしょう」

→ 過去にお金持ちではなかった

仮定法の倒置

仮定法では、ifを省略して倒置することができます。

ifの省略と倒置

仮定法過去の倒置

If I were rich, I would travel.

→ Were I rich, I would travel.

「もし私がお金持ちだったら、旅行するでしょう」

→ ifを省略し、wereを文頭に置く

仮定法過去完了の倒置

If I had studied harder, I would have passed.

→ Had I studied harder, I would have passed.

「もし私がもっと勉強していたら、合格していたでしょう」

→ ifを省略し、hadを文頭に置く

shouldの倒置

If you should need help, please call me.

→ Should you need help, please call me.

「もし助けが必要なら、私に電話してください」

→ 形式的な表現

仮定法の特殊表現

仮定法を使った特殊な表現を覚えましょう。

仮定法の特殊表現一覧

I wish + 仮定法過去（〜だったらいいのに）

I wish I were rich.

「私がお金持ちだったらいいのに」

→ 現在の願望

I wish + 仮定法過去完了（〜だったらよかったのに）

I wish I had studied harder.

「もっと勉強していたらよかったのに」

→ 過去の後悔

as if / as though（まるで〜のように）

He talks as if he were an expert.

「彼はまるで専門家のように話します」

→ 実際は専門家ではない

It's time + 仮定法過去（〜する時だ）

It's time you went to bed.

「もう寝る時です」

→ 仮定法過去を使う

would rather（〜したい）

I would rather you didn't go.

「あなたに行ってほしくない」

→ 仮定法過去を使う

よくある間違いとその理由

仮定法を使う時によくある間違いを見てみましょう。

間違い1：wasとwereを混同する

× 間違い：If I was rich, I would travel.（文法的には可だが、仮定法ではwereが好ましい）

✓ 正しい：If I were rich, I would travel.

理由：仮定法では主語が何でもwereを使うのが正式です（口語ではwasも可）。

間違い2：時制の一致を間違える

× 間違い：If I had studied harder, I would pass the exam.

✓ 正しい：If I had studied harder, I would have passed the exam.

理由：仮定法過去完了の主節も過去完了形にする必要があります。

間違い3：直説法と仮定法を混同する

× 間違い：If it will rain, I will stay home.（未来の可能性がある場合）

✓ 正しい：If it rains, I will stay home.（直説法）

理由：未来の可能性がある場合は直説法を使います。仮定法は現実に反する場合に使います。

間違い4：I wishの後で現在形を使う

× 間違い：I wish I am rich.

✓ 正しい：I wish I were rich.

理由：I wishの後は仮定法過去を使います。

間違い5：倒置の形を間違える

× 間違い：If I had studied, I would have passed.（倒置なし）

✓ 正しい：Had I studied, I would have passed.（倒置）

理由：ifを省略する場合は、hadを文頭に置いて倒置します。

実践練習：仮定法を使おう

仮定法を使った練習をしましょう。

練習1：適切な仮定法を選ぶ

以下の文に適切な仮定法を選びましょう。

1. 「もし私がお金持ちだったら、家を買うでしょう」

✓ If I were rich, I would buy a house.

現在の事実に反するので仮定法過去を使います。

2. 「もし私がもっと勉強していたら、合格していたでしょう」

✓ If I had studied harder, I would have passed.

過去の事実に反するので仮定法過去完了を使います。

3. 「私がお金持ちだったらいいのに」

✓ I wish I were rich.

I wishの後は仮定法過去を使います。

重要ポイントまとめ

1. 仮定法の種類

仮定法過去（現在の事実に反する）、仮定法過去完了（過去の事実に反する）

2. 仮定法過去の構造

If + 過去形, would/could/might + 動詞の原形

3. 仮定法過去完了の構造

If + had + 過去分詞, would/could/might have + 過去分詞

4. wereの使い方

仮定法では主語が何でもwereを使う（口語ではwasも可）

5. would, could, mightの使い分け

would（確実性が高い）、could（可能性・能力）、might（不確実性が高い）

6. 仮定法の倒置

ifを省略してwere/had/shouldを文頭に置く（Were I rich, Had I studied, Should you need）

7. I wish + 仮定法

I wish + 仮定法過去（現在の願望）、I wish + 仮定法過去完了（過去の後悔）

8. as if / as though

「まるで〜のように」という意味で仮定法を使う

9. It's time + 仮定法過去

「〜する時だ」という意味で仮定法過去を使う

10. 直説法と仮定法の違い

直説法は可能性がある場合、仮定法は現実に反する場合に使う
